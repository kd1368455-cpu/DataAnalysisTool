# DataAnalysisTool

## 概要
日常の支払いデータを用いて、業務システムに近いデータ処理を再現した Python プロジェクトです。  
HBA の業務で多い「データ集計」「月次レポート」「支払い手段別の分析」を意識して作成しました。

## 使用技術
- Python
- pandas
- matplotlib

## 機能
- データを DataFrame に変換
- 合計金額・平均金額の算出
- 支払い手段ごとの集計
- 月ごとの集計
- 月次支出の棒グラフ表示

## 工夫した点
- CSV を使わず、コード内にデータを保持することで提出時のファイル消失リスクを回避
- pandas の groupby を用いて業務システムに近い集計処理を再現
- 月次レポートを Period 型で実装し、実務に近い形で分析
- matplotlib による視覚的なレポート生成

## 実行方法
pip install pandas matplotlib
python main.py
