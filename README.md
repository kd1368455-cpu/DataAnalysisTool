# DataAnalysisTool

## 概要
日常の支払いデータを用いて、**業務システムに近いデータ処理フロー**を再現した Python プロジェクトです。  
月次レポート、支払い手段別の集計、可視化を1つのツールで実行できるように設計しています。

本ツールは、**業務系システム開発で求められる「データ処理・分析・可視化」** を意識して構築しました。

---

## 使用技術
- Python  
- pandas  
- matplotlib  
- logging（業務向けログ出力）

---

## 主な機能
### ✔ 基本集計  
- データ件数  
- 合計金額  
- 平均金額  

### ✔ 支払い手段ごとの集計  
- クレカ / 現金 / PayPay などの合計金額を算出  
- 円グラフで割合を可視化  

### ✔ 月ごとの集計  
- Period 型を用いた月次集計  
- 月別の支出推移を算出  

### ✔ グラフの一括表示（1画面に3種類）  
1つのウィンドウに以下をまとめて表示します：  
- 月次棒グラフ  
- 月次折れ線グラフ  
- 支払い手段の円グラフ  

→ **閉じるのは1回だけでOK。視認性が高い業務向けUI。**

### ✔ レポート生成  
- 最も支出が多い月  
- 最も利用された支払い手段  
- 全体の統計まとめ  

→ **業務レポートの形式に近い出力を実現。**

### ✔ 例外処理・ログ出力  
- 日付変換エラーの検知  
- 処理開始/終了ログ  
- 実務で求められる安全性を確保  

---

## 工夫した点
- **3つのグラフを1画面にまとめて表示**し、視認性を向上  
- pandas の groupby を用いて **業務システムに近い集計処理**を再現  
- Period 型で月次レポートを実装し、実務的な分析に対応  
- logging による **業務レベルのログ管理**  
- 例外処理を追加し、データ変換エラーに強い構造  
- コードを関数ごとに分離し、**保守性・再利用性を向上**  
- CSV を使わずコード内にデータを保持し、提出時のファイル消失リスクを回避  

---

## 実行方法

```bash
pip install pandas matplotlib
python main.py
```
